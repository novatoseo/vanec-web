# Prompt para ElevenLabs - Audio VANEC con enfoque Vietnam

Objetivo:
Crear un audio de práctica para acompañar el lead magnet de VANEC - Vanguard English Center. El audio está dirigido a profesionales vietnamitas del sector hotelero que quieren practicar frases esenciales en inglés para atender a huéspedes internacionales.

Idioma del audio:
Principalmente inglés para las frases de práctica, con instrucciones breves en vietnamita al inicio de cada sección.

Estilo de voz recomendado:
- Voz clara, amable y profesional.
- Inglés internacional fácil de entender.
- Acento vietnamita suave o voz pensada para estudiantes vietnamitas.
- No exagerar el acento.
- Debe sonar como un profesor/a vietnamita de inglés con pronunciación clara y profesional.
- Ritmo medio-lento.
- Pausas naturales después de cada frase para que el alumno pueda repetir.
- Tono de hospitalidad: calmado, educado y seguro.

Configuración sugerida en ElevenLabs:
- Voice: Vietnamese-accent English speaker, si está disponible.
- Accent: Vietnamese-accented English, soft and clear.
- Speed: slightly slow.
- Stability: medium-high.
- Similarity: medium.
- Speaker boost: on.
- Emotion: calm, patient, helpful.
- Pauses: leave 2-3 seconds after each phrase.

Instrucciones para la voz:
Read the Vietnamese instructions naturally. Then read each English phrase clearly. After each English phrase, pause so the learner can repeat. Keep a polite hotel-service tone. Do not sound like an advertisement.
